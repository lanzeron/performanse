На цій сторінці допущені наступні помилки:

  • Css, Scripts, jQuery все приєдано в head документу, що сильно погіршить швидкість завантаження сторінки
  • jQuery приєднаний локально, що теж програє в швидкості
  • Картинки використовуються з роширенням png, як в css файлі так і в html
  • присутні теги img з пустими src що робить додаткові запити на сервер, і таким чином погіршує швидкодію
  • пусті поля alt в тегах img
  • запит Ajax з невірною url адресою
